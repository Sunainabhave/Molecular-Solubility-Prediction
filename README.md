# Molecular Solubility Prediction

This project predicts the aqueous solubility (LogS) of molecules using molecular descriptors from the Delaney ESOL dataset. The goal is to compare the performance of different regression models in predicting molecular solubility.

---

## Models Used

- Linear Regression
- Random Forest Regressor

---

## Dataset

- **Delaney ESOL dataset**
- Includes molecular descriptors such as:
  - Molecular weight
  - Number of rotatable bonds
  - Aromatic proportion
  - Target solubility (LogS)
